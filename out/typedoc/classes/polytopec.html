constructor

  • new PolytopeC(elementList: ElementList, constructionRoot?: ConstructionNode<unknown>): PolytopeC
    • Defined in src/ts/polytopes/polytopeTypes.ts:92

    The constructor for the PolytopeC class.

    classdesc

    Represents a polytope as a list of elements, in ascending order of dimensions, similarly (but not identically) to an OFF file. Subelements are stored as indices. All points are assumed to be of the same dimension.

    Parameters

    • elementList: ElementList

      The polytope's element list.

    • Optional constructionRoot: ConstructionNode<unknown>

      The constructionNode representing how the polytope was built.

    Returns PolytopeC
